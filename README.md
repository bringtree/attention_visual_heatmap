##
导入的数据格式:<br/>
真实类别,词1,词2,<br/>
预测类别,词1权重,词2权重,
```xls
calc,根号,76,
video,0.16361079961061478,0.09771940037608147,
calc,根下,25,
video,0.1040461003780365,0.38439180254936217,
```

##
使用方法
打开dist目录下的index.html。然后导入对应格式的xls文件
